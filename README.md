# LED Blink Project

## Objective
Blink an LED every 1 second using Arduino UNO.

## Components Used
- Arduino UNO
- LED
- 220Ω Resistor
- Breadboard

## Concepts Learned
- GPIO Output
- pinMode()
- digitalWrite()
- delay()

## Working
The LED turns ON for 1 second and OFF for 1 second continuously.
